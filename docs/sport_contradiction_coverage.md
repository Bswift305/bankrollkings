# Sport Contradiction Coverage

This file is the cross-sport reminder that every board needs its own contradiction language.

## NBA

- returning-starter squeeze
- role down / minutes down on promoted overs
- zero-required fragile unders
- market gate conflict
- volatility elevation

## WNBA

- trend direction conflict
- zero-required under on low-event stats
- single-book high confidence
- wide book range
- low-sample trend inflation

## MLB

- zero-required hitter unders
- single-book high confidence
- wide line range
- light edge dressed up as conviction
- expensive fragile unders
- trend direction conflict

## NFL

- partial-support featured plays
- tight-support featured plays
- tight-support yardage plays
- low-hit-rate support buckets
- fade-tier plays still scoring too high
- minimum-support sample on featured plays

## CFB

- should eventually have its own set once the live board matures:
  - optional-prop gating
  - roster continuity uncertainty
  - market depth checks
  - game-script dependency

## Operating Principle

When a real miss shows up:

1. notice it
2. name the failure class
3. write the contradiction rule
4. write the archive/replay hook if needed
5. make the same mistake structurally harder next time
