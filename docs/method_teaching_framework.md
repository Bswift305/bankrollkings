# Method Teaching Framework

Bankroll Kings is a methods-first product.

That means we do **not** try to teach every betting idea with the same wording.
We keep one shared teaching structure, then let each method speak in its own language.

## Shared Teaching Slots

Every major method page should explain:

1. `What It Is`
2. `Best Use Case`
3. `What Confirms It`
4. `What Makes It Risky`

This gives the site a consistent learning rhythm without flattening the differences between methods.

## Method Voices

### Floor Plays

- Language focus: stability, role, minutes, lower targets, anchor material
- Emotional tone: calm, disciplined, conservative
- Best for: anchors, safer round-robin builds, avoiding fragile high-variance props

### Trends

- Language focus: streaks, recent form, ride the hot hand, line not catching up
- Emotional tone: active, momentum-driven, opportunistic
- Best for: repeated recent outcomes, especially once a streak reaches 3 or more

### Market Edge

- Language focus: book disagreement, soft numbers, market range, line shopping, DK vs Vegas
- Emotional tone: sharp, selective, price-aware
- Best for: finding value when the market still is not fully aligned

### Matchup Lens

- Language focus: opponent context, pace, spread, total, role fit, injury environment
- Emotional tone: analytical, contextual, confirming or disqualifying
- Best for: deciding whether a trend or edge should actually continue tonight

## Product Rule

Use a shared page structure.
Do **not** use shared method wording.

Good:

- `Floor Plays` talks about stability
- `Trends` talks about streaks
- `Market Edge` talks about pricing
- `Matchup Lens` talks about context

Bad:

- teaching `Trends` with `Floor Play` language
- teaching `Market Edge` like it is just another confidence board
- teaching `Matchup Lens` like it is only a game preview page
